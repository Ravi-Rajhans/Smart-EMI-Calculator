# Smart EMI Calculator

A free and deployable web tool that calculates:

- Monthly EMI
- Total Interest
- Total Payment

## Why this tool

I have personally used EMI calculators when comparing bike/phone purchase options and planning monthly budgets.

## Deploy to Vercel (free)

1. Create a new public repository on GitHub.
2. Push this folder to the repository.
3. Go to https://vercel.com/new and import the GitHub repo.
4. Framework preset: Other (or no framework).
5. Click Deploy.
